# CSS Declarations

- A single statement that comprises of a _property_ and a _value_
- The property is the aspect of the element you want to style, such as `color`, `font-size`, or `margin`
- The value is the specific setting you want to apply to that property, such as `red`, `16px`, or `10px`
- Declarations are written in the format: `property: value;` ending with a semicolon (`;`)
- A [[css-shorthand-properties]] is a declaration that allows you to set multiple properties at once using a single value, such as `margin: 10px;` which sets all four margins (top, right, bottom, left) to `10px`

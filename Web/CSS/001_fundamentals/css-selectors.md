# CSS Selectors

- **universal selector** (`*`) matches any element
- **type/tag selector** matches elements by their tag name, e.g. `p`,
- **class selector** matches elements by their class attribute, e.g. `.text`
- **ID selector** matches elements by their ID attribute, e.g. `#main`
- **attribute selector** matches elements based on the presence or value of an attribute, e.g `[type="text"]`
- **pseudo-class selector** matches elements in a specific state, e.g. `:hover`
- **pseudo-element selector** matches a part of an element, e.g. `::before`

# Syntax

- [[css-declarations]] is a single statement that comprises of a _property_ and a _value_
- Group of declarations is called a **declaration block**
- Declaration blocks are proceeded by **selector(s)**
- A **ruleset**, _ruleset_, is the _declaration block_ together with the [[css-selectors]]
- **at-rules**, i.e. `@import`, are language constructs
